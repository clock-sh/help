# Free Avaiable Services

Clock.sh is only a "Serverless Crontab", it doesn't provide data persistence. But there are some other awesome services you can use! Most of them are free.

## Free Json Store

- [myjson](http://myjson.com)
- [jsonbin](https://jsonbin.io/)
- [jsonstorage](https://jsonstorage.net/)
- [jsonstore](https://www.jsonstore.io/)
- [npoint](https://www.npoint.io/)

## Serverless DB Service

- [fauna](https://fauna.com/)
